## Desafio


Comece instalando a versão 22 do node
```
nvm i
```

## Backend

### Requisitos
- Docker Compose

### Iniciar a Aplicação

Entre na pasta backend
```
cd backend
```

Instale os pacotes

```
npm i
```

Suba os containers da Aplicação e do Banco

```
docker-compose build
docker-compose up
```

## Frontend

### Iniciar a Aplicação

Instale os pacotes

```
npm i
```

Inicie a Aplicação
Instale os pacotes


```
npm run dev
```

Acesse a seguinte URL através do seu navegador

Instale os pacotes

```
http://localhost:3000
```
